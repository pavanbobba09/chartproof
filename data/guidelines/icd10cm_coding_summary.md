# ICD-10-CM coding guidelines (educational summary)

source_id: icd10cm_guidelines_fy2026_summary

Educational demo summary only. Paraphrased for ChartProof rationale letters. Not a complete copy of CMS/CDC official guidelines. Not for clinical use. Not for production coding.

## Section: Principal diagnosis selection

The principal diagnosis is the condition established after study to be chiefly responsible for occasioning the admission. When two or more interrelated conditions meet this definition, sequencing follows the Official Guidelines and any applicable instructional notes.

## Section: Uncertain diagnosis

If the diagnosis documented at the time of discharge is qualified as probable, suspected, likely, questionable, possible, or still to be ruled out, code the condition as if it existed or was established for inpatient admissions, subject to Official Guidelines. Outpatient rules differ; this demo focuses on inpatient.

## Section: Clinical validation context

Clinical validation asks whether clinical indicators in the chart support a documented diagnosis that affects payment grouping. A diagnosis listed only in a billing query without supporting clinical indicators may be challenged. ChartProof drafts cite chart lines and criteria summaries; a human auditor decides.

## Section: Sepsis coding notes (high level)

Sepsis coding depends on provider documentation and applicable guideline updates for the code year. R65.2x codes relate to severe sepsis when criteria and documentation support them. This demo does not implement full official sequencing trees; it only checks simplified clinical support for a billed sepsis diagnosis.
